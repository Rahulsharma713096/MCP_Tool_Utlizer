# 2. Single Command Execution Design

---

# Windows

## `run.bat`

```bat id="d0hjnr"
@echo off

echo Starting AI Workspace...

call scripts\check_env.bat

if %ERRORLEVEL% neq 0 (
    echo Environment validation failed
    pause
    exit /b
)

start cmd /k "cd apps\backend && python main.py"

timeout /t 5

start cmd /k "cd apps\frontend && npm run dev"

echo AI Workspace Started
pause
```

---

# Linux/Mac

## `run.sh`

```bash id="zj99zv"
#!/bin/bash

echo "Starting AI Workspace"

bash scripts/check_env.sh

if [ $? -ne 0 ]; then
    echo "Environment validation failed"
    exit 1
fi

cd apps/backend && python main.py &

sleep 5

cd ../frontend && npm run dev
```

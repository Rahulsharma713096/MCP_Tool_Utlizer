# Advanced AI Workspace Platform — Full Architecture & Project Design

You are essentially building a production-grade **AI Operations Workspace**:

A local-first desktop/web application where users can:

* Manage local LLMs via Ollama
* Run/stop models dynamically
* Manage MCP servers/tools from UI
* Chat with selected model
* Connect online providers (OpenRouter, Gemini, Z.ai, OpenAI, Claude, etc.)
* Dynamically switch providers/models
* Monitor runtime/CPU/RAM/GPU
* Kill inactive model processes
* Auto-detect installations
* Configure providers via UI
* Hot reload configs
* Plugin-based architecture
* Enterprise-level UX/UI

# Major Modules

# 1. Ollama Runtime Manager

This is the heart of local model control.

---

## Features

### Detect Ollama Installation

Backend checks:

```bash
ollama --version
```

If not installed:

UI shows:

* Install button
* OS-specific guide
* Download link
* Auto setup assistant

---

## Fetch Installed Models

Command:

```bash
ollama list
```

UI shows:

* model name
* size
* quantization
* runtime state
* VRAM usage
* CPU usage

---

## Start Model Runtime

```bash
ollama run llama3
```

OR API:

```bash
POST /api/generate
```

---

## Kill Running Models

Critical feature.

Use:

```python
psutil.process_iter()
```

Find:

```text
ollama.exe
```

Then:

```python
process.kill()
```

---

## Multi-Model Runtime Control

User selects:

```text
[✓] llama3
[ ] mistral
[ ] deepseek
```

System:

* starts selected model
* unloads inactive models
* frees RAM/VRAM

---

## Runtime Monitoring

Live metrics:

* CPU %
* GPU %
* RAM
* VRAM
* tokens/sec
* latency
* active sessions

# 8. Database Design

# SQLite Schema

## models

```sql
CREATE TABLE models (
    id INTEGER PRIMARY KEY,
    name TEXT,
    provider TEXT,
    active BOOLEAN,
    runtime_pid INTEGER
);
```

---

## mcps

```sql
CREATE TABLE mcps (
    id INTEGER PRIMARY KEY,
    name TEXT,
    enabled BOOLEAN,
    config JSON
);
```

---

## chats

```sql
CREATE TABLE chats (
    id INTEGER PRIMARY KEY,
    session_id TEXT,
    role TEXT,
    content TEXT,
    timestamp DATETIME
);
```
